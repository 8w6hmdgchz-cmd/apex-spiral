## 一、神经网络核心机制

### 1. 神经元结构
```
树突(dendrite) → 胞体(soma) → 轴突(axon) → 突触(synapse)
```
- **树突**: 接收输入信号
- **胞体**: 信息整合处理
- **轴突**: 信号传导
- **突触**: 信号传递到下一个神经元

### 2. 动作电位 (Action Potential)
```
静息电位: -70mV
去极化: Na+通道打开，→ +30mV
复极化: K+通道打开，恢复
超极化: 短暂过负
```
**公式:**
```
V_m(t) = V_rest + I_syn(t) · R_m
```

### 3. 突触可塑性 (Synaptic Plasticity)
```
ΔW_syn = η · (V_m - V_th) · e^(-β·Δt)
```
- **LTP (长时程增强)**: 高频刺激 → 突触强化
- **LTD (长时程抑制)**: 低频刺激 → 突触弱化
- **Hebbian规则**: "一起激活的神经元，连接加强"

### 4. 神经编码
| 编码方式 | 描述 | AI对应 |
|----------|------|--------|
| 频率编码 | 放电频率传递信息 | 激活值大小 |
| 时间编码 | 精确放电时间 | Transformer位置编码 |
| 群体编码 | 神经元群体协同 | Multi-Head Attention |

## 二、神经网络训练机制

### 1. 前向传播
```
Input → Linear → Activation → Linear → Output
x → Wx+b → σ() → y
```

### 2. 损失函数
```
L = (1/N) · Σ(y_true - y_pred)²
```

### 3. 反向传播 (Backpropagation)
```
∂L/∂W = ∂L/∂y · ∂y/∂z · ∂z/∂W
δ_l = σ'(z_l) · W_{l+1}ᵀ · δ_{l+1}
```
**链式法则:** 误差从输出层向输入层传播

### 4. 梯度下降
```
W_new = W_old - α · ∂L/∂W
```

| 离子通道 | Nav/Kv/Cav通道 | ★★ |

### 3. 全细胞模拟公式

#### 3.1 膜电位动力学 (Hodgkin-Huxley)
```
I = C_m · dV/dt + g_K·n⁴·(V-V_K) + g_Na·m³·h·(V-V_Na) + g_L·(V-V_L)

dn/dt = α_n·(1-n) - β_n·n
dm/dt = α_m·(1-m) - β_m·m
dh/dt = α_h·(1-h) - β_h·h
```
- **I**: 注入电流
- **C_m**: 电容
- **g_K, g_Na, g_L**: 电导
- **n, m, h**: 门变量

#### 3.2 化学反应动力学
```
d[X]/dt = Σ(v_production) - Σ(v_consumption)
v = k · [substrate]
```

#### 3.3 扩散方程
```
∂C/∂t = D · ∇²C
```
- **C**: 浓度
- **D**: 扩散系数
