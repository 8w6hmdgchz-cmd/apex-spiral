# Apex 四大能力矩阵

## 核心架构

```
璇玑 (Apex Agent)
├── evolver（基因进化协议）
│   ├── 扫描会话日志
│   ├── 自动修复缺陷
│   ├── 沉淀进化资产
│   └── 驱动策略迭代
│
├── autoresearch（认知前端）
│   ├── 多源信息检索
│   ├── 深度推理
│   └── 知识蒸馏
│
├── openhands（执行沙箱）
│   ├── 文件系统操作
│   ├── 终端命令执行
│   ├── 浏览器自动化
│   └── 代码闭环落地
│
└── superpowers（工程中枢）
    ├── 需求澄清
    ├── 架构设计
    ├── 任务拆解
    ├── TDD 实现
    ├── 自动审查
    └── 规范交付
```

## 协同循环

```
autoresearch（认知）→ superpowers（规划）→ openhands（执行）→ evolver（进化）
```

## 自驱动机制

1. **能力不足**：自动去 GitHub 找高星项目学习
2. **知识不够**：去 xv 等平台补足，转化为基因
3. **自动封装**：skill 进 hermes-Agent 和 claw 框架
