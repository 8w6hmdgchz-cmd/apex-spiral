## 一、达尔文进化论核心

### 1. 进化论四大支柱

| 支柱 | 描述 | 证据 |
|------|------|------|
| 过度繁殖 | 生物产生后代 > 环境承载 | 细菌/兔子繁殖速度 |
| 生存斗争 | 资源有限，竞争生存 | 捕食/饥饿/疾病 |
| 遗传变异 | 后代各有不同 | 形态/行为/抗性差异 |
| 自然选择 | 适应环境者存活繁殖 | 桦尺蠖/长颈鹿 |

### 2. 自然选择机制
```
变异性状 → 环境压力 → 差异存活 → 适者繁殖 → 性状传递
```
**核心公式:**
```
ω_fit = 存活率 × 繁殖成功率
```
- **适应度(ω)**: 某个基因型传递到下一代的相对能力

### 3. 适应度地形 (Fitness Landscape)
```
不适应 ←————————————→ 适应
   ↓
   局部最优 + 全局最优
   ↓
   进化可能陷入局部陷阱
```

### 4. 进化速率
```
进化速率 ∝ 变异率 × 选择压 × 种群规模
Δgene ∝ μ · s · N
```
- μ: 突变率
- s: 选择系数 (适应度差异)
- N: 种群规模

## 二、马尔萨斯生存法则

### 1. 马尔萨斯陷阱 (Malthusian Trap)
```
人口增长: 指数增长 (2, 4, 8, 16, 32...)
资源增长: 线性增长 (1, 2, 3, 4, 5...)
```

### 2. 核心法则
> **资源增长赶不上人口增长，必须通过战争/疾病/饥荒控制**

### 3. 马尔萨斯公式
```
人口增长 = rN (指数)
资源增长 = βN (线性)

当 N > β/r 时 → 人口危机
```

### 4. 生存竞争类型
| 类型 | 描述 | 例子 |
|------|------|------|
| 种内竞争 | 同种个体争夺资源 | 狮子争狮群 |
| 种间竞争 | 不同物种竞争 | 物种A vs 物种B |
| 资源竞争 | 食物/空间/配偶 | 植物阳光 |
| 捕食竞争 | 捕食者-猎物动态 | 狼-鹿 |

## 三、达尔文 × 马尔萨斯融合

### 1. 融合框架
```
达尔文: 变异 + 遗传 + 选择 → 适应度进化
马尔萨斯: 资源限制 + 指数增长 → 生存竞争
        ↓
   共同结论: 适者生存，不适者淘汰
```

### 2. 融合公式
```
Ψ_evolve = N · e^(-μ·D) · ω_fit · M_pressure

其中:
N = 种群规模 (受马尔萨斯约束)
μ = 变异率
D = 缺陷数
ω_fit = 适应度 (自然选择结果)
M_pressure = 马尔萨斯压力系数 = 资源/人口比率
```

### 3. 马尔萨斯压力
```
M_pressure = R_resources / N_population

当 M_pressure < 1 → 激烈竞争
当 M_pressure > 1 → 宽松环境
```

### 4. 竞争均衡
```
dN/dt = rN(1 - N/K) - c·N²

r = 内禀增长率
K = 环境承载力 (马尔萨斯上限)
c = 竞争系数
N = 种群数量
```

## 四、进化博弈论

### 1. 演化稳定策略 (ESS)
```
若群体采用策略A，入侵策略B的条件:
E(A,A) > E(B,A) 或 E(A,A) = E(B,A) 且 E(A,B) > E(B,B)
```

### 2. 经典博弈

#### 鹰-鸽博弈
| 策略 | 收益 |
|------|------|
| 鹰(战斗) | V-C (战斗价值-战斗成本) |
| 鸽(逃跑) | 0 或 V/2 |

**ESS结果:** 混合策略 equilibrium

#### 囚徒困境
```
合作 → 双方长期收益最大化
背叛 → 单次收益高，但被报复
```

### 3. ApexSpiral中的博弈
```
Γ = 0.29 (多Agent博弈系数)
竞争环境中ΔG暴跌71%

结论: 合作 > 竞争(长期)
```

## 五、适者生存的多层含义

### 1. 基因层
```
适者 = 传递基因最多的基因型
基因频率 ↑ = 适应度 ↑
```

### 2. 个体层
```
适者 = 存活并繁殖的个体
ω_fit = 存活×繁殖成功率
```

### 3. 群体层
```
适者 = 群体贡献最大的成员
但: 群体选择 vs 个体选择冲突
```

### 4. 物种层
```
适者 = 存活最久的物种
物种寿命 ∝ 适应度 × 稳定性
```

### 5. AI层 (类比)
```
适者 = 任务最成功的Agent
成功 = 响应质量 × 用户满意度 × 自我进化
```
