# 肿瘤免疫治疗进展

## 一、免疫检查点抑制剂（ICI）

### 核心机制
肿瘤细胞利用免疫检查点通路逃避免疫监视。ICI阻断这一通路，恢复T细胞对肿瘤的杀伤能力。

### 主要靶点

| 靶点 | 药物 | 适应症 |
|------|------|---------|
| **PD-1** | Pembrolizumab, Nivolumab | 黑色素瘤、NSCLC、膀胱癌等 |
| **PD-L1** | Atezolizumab, Durvalumab | 膀胱癌、NSCLC |
| **CTLA-4** | Ipilimumab | 黑色素瘤 |

### 疗效预测标志物

**PD-L1表达**
- IHC检测
- 阳性阈值：TPS ≥1%（Keytruda）、≥50%（高表达）
- 但不是完美预测标志物（PD-L1阴性也有效）

**TMB（肿瘤突变负荷）**
- 高TMB → 更多新抗原 → 更强免疫应答
- FoundationOne等商业检测
- 争议：最佳阈值未确定

**MSI-H/dMMR**
- 微卫星高度不稳定 = 免疫治疗强响应
- 泛癌种适应症

## 二、CAR-T细胞治疗

### 结构
```
CAR = 抗原结合域（ScFv）+ 铰链区 + 跨膜域 + 共刺激信号（CD28/4-1BB）+ CD3ζ信号域
```

### 迭代

| 迭代 | 特点 | 代表 |
|------|------|------|
| 第一代 | CD3ζ alone | 体内扩增差 |
| 第二代 | +共刺激（CD28或4-1BB）| Yescarta, Kymriah |
| 第三代 | +多个共刺激 | 临床中 |
| 第四代 | +转基因（IL-12等）| 临床中 |

### 适应症

| 药物 | 靶点 | 适应症 |
|------|------|---------|
| **Kymriah** | CD19 | ALL(儿童/青年), DLBCL |
| **Yescarta** | CD19 | DLBCL, FL |
| **Breyanzi** | CD19 | LBCL |
| **Abecma** | BCMA | 多发性骨髓瘤 |

### 毒性

**CRS（细胞因子释放综合征）**
- IL-6介导
- 分级：轻度（发热）→ 重度（器官衰竭）
- 治疗：Tocilizumab（IL-6R抗体）+ 糖皮质激素

**ICANS（免疫效应细胞相关神经毒性）**
- CD19靶点CAR-T常见
- 意识障碍、癫痫
- 监测神志变化

## 三、免疫联合治疗

### 联合策略

| 组合 | 机制 | 证据 |
|------|------|------|
| ICI + ICI | PD-1 + CTLA-4 | Nivo+Ipi黑色素瘤 |
| ICI + 化疗 | 化疗→抗原释放+ICI | 多种实体瘤 |
| ICI + 抗血管 | 改善免疫微环境 | 肾癌、肝癌 |
| ICI + 放疗 | 远隔效应（abscopal）| 临床研究中 |

### 挑战
- 毒性叠加
- 最佳顺序未知
- 费用高昂

## 四、免疫耐药机制

### 原发性耐药（ intrinsic resistance）
- IFN-γ信号通路缺陷
- 抗原呈递缺陷（B2M突变）
- WNT/β-catenin激活
- PTEN丢失

### 获得性耐药（ acquired resistance）
- JAK1/2突变 → IFN-γ不响应
- B2M突变 → MHC-I表达丢失
- 新抗原丢失

### 克服策略
- 靶向克服耐药通路
- 新型检查点（TIGIT, LAG-3, TIM-3）
- 肿瘤疫苗
- 双特异性抗体

---

## 五、过继性细胞治疗（ACT）

### TIL（肿瘤浸润淋巴细胞）
- 从肿瘤组织提取T细胞，体外扩增，回输
- 黑色素瘤效果显著
- 其他实体瘤探索中

### TCR-T
- T细胞受体基因工程改造
- 识别肿瘤抗原（NY-ESO-1等）
- 优于CAR-T：可识别胞内抗原

### NK细胞治疗
- 异体NK细胞安全（无GVHD）
- CAR-NK：结合CAR靶向+NK天然杀伤
- iPSC来源NK细胞

## 六、肿瘤疫苗

### 新抗原疫苗
- 测序肿瘤突变→预测新抗原→合成个性化疫苗
- mRNA疫苗（BioNTech/Moderna）+多肽疫苗
- 2023年：黑色素瘤III期成功

### 治疗性疫苗
| 类型 | 例子 |
|------|------|
| 树突细胞疫苗 | Sipuleucel-T（前列腺癌）|
| 病毒载体 | T-VEC（黑色素瘤，HSV）|

## 七、双特异性抗体

### T细胞衔接器（BiTE）
```
CD3结合域 + 肿瘤抗原结合域
同时结合T细胞和肿瘤细胞 → T细胞杀伤
```

| 药物 | 靶点 | 适应症 |
|------|------|---------|
| **Blinatumomab** | CD19×CD3 | ALL |
| **Mosunetuzumab** | CD20×CD3 | FL |
| **Elranatamab** | BCMA×CD3 | 多发性骨髓瘤 |

---

## 八、免疫相关不良反应（irAE）

### 常见irAE
| 系统 | 表现 | 处理 |
|------|------|------|
| 皮肤 | 皮疹、瘙痒 | 激素外用 |
| 胃肠道 | 结肠炎、腹泻 | 英夫利昔/激素 |
| 肝脏 | 肝炎（转氨酶↑）| 激素 |
| 内分泌 | 甲状腺炎、肾上腺炎 | 激素替代 |
| 肺 | 肺炎（免疫性肺炎）| 激素，暂停ICI |

### 管理原则
- 轻症：糖皮质激素外用/口服
- 重症：静脉激素 + 考虑ICI永久停用
- 英夫利昔单抗（激素不改善时）

## 九、未来方向

1. **新靶点**：TIGIT, LAG-3, TIM-3, VISTA
2. **精准筛选**：多标志物联合预测模型
3. **克服耐药**：ADC+免疫、靶向+免疫
4. **CAR-T实体瘤突破**：靶点选择、肿瘤微环境改造
5. **mRNA疫苗平台**：个性化新抗原疫苗

## 十、批判性评价要点

**临床试验设计问题：**
- ICI的替代终点（ORR/PFS）能否预测OS？
- 免疫治疗拖尾效应——PFS曲线尾巴不代表长期获益
- 交叉设计允许对照组患者后线使用ICI → OS分析需调整
- 免疫治疗适合的统计方法：landmark分析、time-varying分析

**疗效评价问题：**
- irAE vs 肿瘤进展（假性进展pseudoprogression）
- iRECIST标准 vs RECIST 1.1
