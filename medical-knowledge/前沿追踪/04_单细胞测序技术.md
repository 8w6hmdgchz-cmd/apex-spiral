# 单细胞测序技术

## 一、技术原理

### 核心概念
传统bulk RNA-seq = 混合细胞平均信号  
单细胞RNA-seq = 每个细胞的基因表达谱

### 技术流程
```
组织 → 单细胞悬液 → 单细胞捕获 → 建库 → 测序 → 数据分析
                              ↓
                    细胞barcode + UMI（唯一分子标识符）
```

### 捕获技术

| 技术 | 原理 | 优缺点 |
|------|------|---------|
| **10x Genomics** | 油包水微滴（Droplet）| 通量高，成本低 |
| **Smart-seq2** | Plate-based | 全长转录本，敏感 |
| **sci-RNA-seq** | 组合索引 | 超低成本 |
| **Microwell-seq** | 微孔板 | 成本低 |

## 二、10x Genomics Chromium

### 原理
```
油包水微滴（Droplet）
细胞 + bead（含barcode） → 同一微滴
        ↓
逆转录 → cDNA → 建库 → 测序
```

### barcode结构
```
10x Barcode = 16bp细胞barcode + 10bp UMI
每个细胞唯一 → 区分来源
UMI = 标记原始分子 → 定量
```

### 读长结构
```
Read 1: 细胞barcode + UMI
Read 2: 转录本序列
i7/i5: 样本index
```

## 三、数据分析流程

### 基础分析
```bash
# Cell Ranger（10x官方）
cellranger count --id=sample \
  --transcriptome=/refdata-gex-GRCh38 \
  --fastqs=/path/to/fastq \
  --sample=sample
```

### R分析流程
```r
library(Seurat)

# 导入数据
data <- Read10X(data.dir = "/filtered_feature_bc_matrix/")
seurat_obj <- CreateSeuratObject(counts = data, project = "sample")

# QC
seurat_obj <- subset(seurat_obj, 
                      subset = nFeature_RNA > 200 & 
                              nFeature_RNA < 5000 & 
                              percent.mt < 10)

# 标准化
seurat_obj <- NormalizeData(seurat_obj)
seurat_obj <- FindVariableFeatures(seurat_obj, nfeatures = 2000)
seurat_obj <- ScaleData(seurat_obj)

# 降维
seurat_obj <- RunPCA(seurat_obj)
seurat_obj <- RunUMAP(seurat_obj, dims = 1:30)

# 聚类
seurat_obj <- FindNeighbors(seurat_obj, dims = 1:30)
seurat_obj <- FindClusters(seurat_obj, resolution = 0.5)
```

## 四、降维与聚类

### 降维方法

| 方法 | 特点 |
|------|------|
| **PCA** | 线性降维，计算快 |
| **t-SNE** | 非线性，保留局部结构，保留全局差 |
| **UMAP** | 快，保留更多全局结构，推荐 |

### 聚类方法
```r
# KNN图 + Louvain算法
seurat_obj <- FindClusters(seurat_obj, algorithm = 1)  # Louvain

# Leiden算法（更优）
seurat_obj <- FindClusters(seurat_obj, algorithm = 4)  # Leiden
```

### 分辨率选择
- 分辨率越高 → 亚群越多
- 需要生物学意义指导
- 0.4-0.8常用于一般分析
- 低分辨率：粗分类；高分辨率：探索亚群

## 九、常见问题

| 问题 | 原因 | 处理 |
|------|------|------|
| 细胞破碎 | 组织解离过度 | 优化解离条件 |
| 双细胞 | 细胞密度高 | doublet检测，移除 |
| 细胞死亡 | 解离应激 | 凋亡指标过滤（percent.mt）|
| 批次效应 | 不同批次 | Harmony/MNN/Batchkmeans校正 |

## 十、批判性评价要点

**实验设计问题：**
- 样本量：单细胞≠不需要重复
- 组织保存：新鲜vs固定vs冷冻
- 解离方案：不同组织不同

**数据分析问题：**
- 聚类主观性（不同分辨率不同结果）
- 细胞注释依赖数据库完整性
- Dropout（零 inflation）处理

**结论推断问题：**
- 相关性≠因果
- 体外/in vivo差异
- 扰动实验验证必要性
