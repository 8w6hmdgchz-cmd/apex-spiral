# 基因编辑（CRISPR）临床应用

## 一、CRISPR-Cas9原理

### 核心组件
```
Guide RNA (gRNA) + Cas9蛋白
         ↓
gRNA引导Cas9定位到目标DNA
         ↓
Cas9切割DNA双链（DSB）
         ↓
细胞HDR/NHEJ修复 → 基因敲除/敲入
```

### Cas9变体
| 蛋白 | 切割方式 | 应用 |
|------|----------|------|
| **Cas9 WT** | 双链断裂（DSB）| 敲除 |
| **Cas9 nickase** | 单链切割 | 减少脱靶 |
| **Cas9 D10A** | nickase突变体 | 高保真 |
| **dCas9** | 无切割活性 | 转录调控/成像 |

### HDR vs NHEJ
- **HDR（同源重组修复）**：精准敲入，需供体模板，效率低，仅分裂期细胞
- **NHEJ（非同源末端连接）：** 随机插入/缺失，效率高，适合敲除

## 二、递送系统

### 病毒递送
| 载体 | 特点 | 临床应用 |
|------|--------|----------|
| **AAV** | 无致病性，容量~4.7kb，免疫原性低 | 眼科、血液病 |
| **慢病毒** | 整合基因组，长期表达 | CAR-T，体外基因治疗 |
| **腺病毒** | 高容量，免疫原性强 | 肿瘤（溶瘤病毒）|

### 非病毒递送
| 方法 | 特点 |
|------|------|
| **LNP（脂质纳米粒）** | mRNA递送主流，安全，可重复 |
| **电转** | 体外高效率，体内困难 |
| **金纳米颗粒** | 研究中 |
| **RNP（核糖核蛋白）** | Cas9蛋白+gRNA复合物，瞬时表达，脱靶低 |

## 四、热门研究方向

### 肿瘤免疫治疗
- **CAR-T + CRISPR**：敲除PD-1/TIM-3/FLT3增强CAR-T
- **CRISPR筛选**：发现新靶点、耐药机制
- **基因敲入增强**：CAR基因精准插入特定位点

### 遗传病
- **镰状细胞病/β地中海**：BCL11A增强子敲除→恢复胎儿血红蛋白
- **杜氏肌营养不良**：外显子跳跃矫正
- **先天性失明**：CEP290基因编辑（Leber先天性黑矇）

### 传染病
- **HIV**：CCR5敲除（功能性治愈）
- **HBV**：HBV DNA降解

## 五、脱靶效应与安全性

### 脱靶检测
| 方法 | 原理 |
|------|------|
| **GUIDE-seq** | 双链寡核苷酸捕获 |
| **DISCOVER-seq** | MRE11招募 |
| **CIRCLE-seq** | 体外捕获 |
| **CHANGE-seq** | 接头连接 |
| **TARGET-seq** | 敏感检测 |

### 安全性问题
- **脱靶切割**：非目标位点的突变
- **大片段缺失**：Cas9切割位点附近
- **染色体截断**：DMD基因编辑时观察到
- **p53激活**：Cas9表达→p53激活→细胞生长抑制

### 应对策略
- 高保真Cas9变体（eSpCas9, HypaCas9）
- 缩短gRNA
- RNP递送（瞬时表达）
- 体外筛选脱靶克隆

## 六、碱基编辑与先导编辑

### 碱基编辑（Base Editing）
- **胞嘧啶碱基编辑（CBE）**：C→T（AID/ APOBEC脱氨酶）
- **腺嘌呤碱基编辑（ABE）**：A→G（TadA脱氨酶）
- **无DNA双链断裂**：更安全
- **PE（先导编辑）**：精准插入/删除/转换

### 优势
- 无DSB → 无大片段缺失/染色体截断
- 无需供体模板
- 效率高于HDR

## 七、监管与伦理

### FDA/EMA批准要求
- 详细的脱靶分析
- 全基因组测序（WGS）评估
- 长期随访（15年）
- CMC（制造质量控制）

### 伦理争议
- **体细胞编辑**：可接受
- **生殖系编辑**：禁止（国际共识）
- **增强性编辑**：争议大

---

## 八、批判性评价要点

**临床前研究常见问题：**
- 体外实验 → 体内转化效率低
- 小鼠模型 → 人体安全性未验证
- 脱靶检测方法灵敏度不足

**临床研究中问题：**
- 递送效率（in vivo仍是瓶颈）
- 持久性（vs AAV的长期表达需求）
- 成本（个性化治疗价格昂贵）
