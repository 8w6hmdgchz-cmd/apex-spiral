# 细胞生物学基础

## 一、细胞基本结构

### 真核细胞 vs 原核细胞

| 结构 | 真核细胞 | 原核细胞 |
|------|----------|----------|
| 细胞核 | 有核膜 | 无（拟核区） |
| DNA | 线性，多条染色体 | 环形，单一 |
| 线粒体 | 有 | 无 |
| 核糖体 | 80S | 70S |
| 大小 | 10-100μm | 1-5μm |

## 二、细胞膜

### 流动镶嵌模型
```
磷脂双分子层 + 蛋白质 + 胆固醇 + 糖链
```

### 跨膜物质转运

| 方式 | 特点 | 例子 |
|------|------|------|
| **单纯扩散** | 顺浓度，无需能量 | O₂, CO₂, 脂溶性物质 |
| **易化扩散** | 顺浓度，需载体/通道 | 葡萄糖转运蛋白 |
| **主动转运** | 逆浓度，耗ATP | Na⁺/K⁺-ATPase |
| **胞吞/胞吐** | 大分子 | 蛋白质、脂蛋白 |

### 主动转运类型
- **原发性主动转运**：直接耗ATP（如Na⁺/K⁺泵）
- **继发性主动转运**：利用电化学梯度（如葡萄糖-钠同向转运）

## 三、细胞器功能

### 线粒体
- 有氧呼吸场所（氧化磷酸化）
- ATP合成：F₀F₁-ATP合酶
- 半自主细胞器（有独立DNA）
- 参与凋亡（释放细胞色素C）

### 内质网（ER）
- **粗面ER**：核糖体附着，蛋白质合成与折叠
- **滑面ER**：脂质合成，药物代谢，Ca²⁺储存

### 高尔基体
- 蛋白质加工、修饰（糖基化）
- 蛋白质分选与包装
- 溶酶体形成

### 溶酶体
- 酸性水解酶
- 细胞内消化（自噬、异噬）
- 参与细胞凋亡

---

## 四、细胞骨架

### 微管
- α/β-微管蛋白二聚体组成
- 中心粒、纤毛、鞭毛结构
- 物质运输（微管马达蛋白： kinesin/dynein）
- 有丝分裂时形成纺锤体
- 秋水仙碱抑制聚合，紫杉酚促进稳定

### 微丝（肌动蛋白）
- G-actin → F-actin（聚合）
- 细胞运动、胞质分裂、肌肉收缩
- 细胞骨架重构（迁移细胞前缘）
- **Cytochalasin** 抑制微丝

### 中间纤维
- 6类（角蛋白、波形蛋白、神经纤维等）
- 提供机械支撑
- 细胞身份标志物（病理诊断基础）

## 五、细胞周期

### 分期
```
G₁期（合成前期）→ S期（DNA合成）→ G₂期（合成后期）→ M期（分裂）
                            ↑
                       G₀期（静息期）
```

### Checkpoint
- **G₁/S检查点**：细胞是否进入S期（DNA损伤检查）
- **G₂/M检查点**：DNA复制是否完成
- **M期中后期检查点**：纺锤体装配检查

### 调控分子
| 分子 | 功能 |
|------|------|
| Cyclin | 周期性表达，结合CDK |
| CDK | 蛋白激酶，磷酸化底物 |
| CDK抑制剂（CKI）| 负调控 |
| p53 | 抑癌基因，G₁/S检查点核心 |
| Rb | 抑癌基因，限制E2F转录因子 |

### 细胞周期与肿瘤
- p53突变 → 失去G₁/S检查点 → 基因组不稳定
- Rb突变 → E2F持续激活 → 过度增殖
- CDK4/6过表达 → 加速G₁→S

## 六、细胞死亡

### 凋亡（Apoptosis）
**特征：** 细胞皱缩、染色质浓缩、DNA片段化、凋亡小体、被吞噬

**外源性途径：** 死亡受体（Fas/TNFR）→ caspase-8 → caspase-3
**内源性途径：** 线粒体释放细胞色素C → Apaf-1 → caspase-9 → caspase-3

### 坏死（Necrosis）
- 细胞肿胀、膜破裂、内容物释放
- 炎症反应
- **程序性坏死（Necroptosis）：** RIPK1/3介导，炎症性死亡

### 自噬（Autophagy）
- 细胞降解自身受损细胞器
- 回收原料（氨基酸、脂肪酸）
- 依赖溶酶体
- 适度自噬：细胞保护；过度自噬：自噬性死亡

## 七、细胞信号转导

### 受体类型

| 类型 | 位置 | 例子 |
|------|------|------|
| **G蛋白偶联受体（GPCR）** | 细胞膜 | 肾上腺素受体、嗅觉受体 |
| **酪氨酸激酶受体（RTK）** | 细胞膜 | 胰岛素受体、EGFR |
| **离子通道型受体** | 细胞膜 | 乙酰胆碱受体（NMJ） |
| **核受体** | 细胞内 | 甾体激素受体、甲状腺激素受体 |

### 第二信使
- cAMP（腺苷酸环化酶）
- IP₃/DAG（磷脂酶C）
- Ca²⁺
- NO

### 关键信号通路
**PI3K/AKT/mTOR：** 促进细胞生长、增殖、存活
**MAPK/ERK：** 细胞增殖、分化
**JAK/STAT：** 细胞因子信号

## 九、细胞分化与干细胞

### 干细胞特性
- 自我更新（分裂后仍是干细胞）
- 多向分化潜能

### 胚胎干细胞（ESC）
- 全能性（可分化为任何细胞类型）
- 伦理争议

### 成体干细胞
- 多能性（如造血干细胞、间充质干细胞）
- 组织修复
