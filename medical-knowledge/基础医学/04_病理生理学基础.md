# 病理生理学基础

## 一、基本概念

### 病理生理学定义
研究疾病发生、发展和转归的规律，揭示疾病本质的科学。

### 病因分类
| 类型 | 举例 |
|------|------|
| 生物因素 | 病原微生物、寄生虫 |
| 物理因素 | 创伤、辐射 |
| 化学因素 | 毒素、药物 |
| 遗传因素 | 基因突变、染色体异常 |
| 营养因素 | 缺乏、过剩 |
| 免疫因素 | 超敏反应、自身免疫 |

## 二、细胞损伤

### 损伤原因
- 缺氧（最常见）
- 物理化学因素
- 生物因素
- 营养因素
- 免疫因素

### 损伤机制
```
缺氧 → 线粒体功能障碍 → ATP↓ → 细胞损伤
         ↓
钠泵功能障碍 → 细胞水肿
钙泵功能障碍 → 钙超载 → 激活磷脂酶/蛋白酶
```

### 可逆性 vs 不可逆损伤
**可逆性：** 细胞水肿、脂肪变性  
**不可逆：** 严重线粒体损伤、膜完整性丧失、溶酶体破裂

## 三、水盐代谢

### 水钠平衡
```
细胞外液（ECF）≈ 20% 体重
     ↓
血浆 ≈ 5%，组织间液 ≈ 15%
```

### 水钠代谢紊乱

| 类型 | 血浆Na⁺ | 血浆渗透压 | 细胞外液 |
|------|----------|------------|----------|
| **低渗性脱水** | ↓ | ↓ | ↓ |
| **高渗性脱水** | 正常/↑ | ↑ | ↓ |
| **等渗性脱水** | 正常 | 正常 | ↓ |
| **水中毒** | ↓ | ↓ | ↑ |

### 水肿机制
- 血管内外流体静压差↑
- 血浆胶体渗透压↓
- 血管通透性↑
- 淋巴回流障碍

## 四、酸碱平衡

### 缓冲系统
| 缓冲对 | 比例 | 作用 |
|--------|------|------|
| **HCO₃⁻/H₂CO₃** | 20:1 | 最重要的缓冲对 |
| 蛋白质 | — | 细胞内缓冲 |
| 磷酸盐 | — | 细胞内缓冲 |
| 血红蛋白 | — | 红细胞内缓冲 |

### 酸碱失衡

**代谢性酸中毒**
```
原因：乳酸酸中毒、酮症、肾衰竭、腹泻
代偿：H⁺ + HCO₃⁻ → H₂CO₃ → CO₂ + H₂O
      呼吸代偿：呼吸加深加快 → PaCO₂ ↓
```

**呼吸性酸中毒**
```
原因：COPD、呼吸中枢抑制
代偿：肾代偿：H⁺ + HCO₃⁻ → 保留碱储
```

### 常用指标
| 指标 | 正常值 | 意义 |
|------|---------|------|
| pH | 7.35-7.45 | H⁺浓度 |
| PaCO₂ | 35-45 mmHg | 呼吸性酸碱 |
| HCO₃⁻ | 22-26 mmol/L | 代谢性酸碱 |

### 阴离子间隙（AG）
```
AG = Na⁺ - (Cl⁻ + HCO₃⁻)
正常：8-12 mmol/L
AG↑ → 有机酸堆积（乳酸、酮体）
```

## 五、发热

### 机制
```
外源性致热原（ LPS） → 激活单核/巨噬细胞
         ↓
内源性致热原（IL-1、TNF、IL-6） → 下丘脑POAH
         ↓
体温调定点上移 → 发热
```

### 发热时相
| 时相 | 特点 |
|------|------|
| **体温上升期** | 产热>散热，皮肤血管收缩→苍白、寒战 |
| **高热持续期** | 产热=散热，皮肤血管扩张→发热 |
| **体温下降期** | 散热>产热，出汗、退热 |

### 热型
| 类型 | 特点 | 疾病 |
|------|------|------|
| 稽留热 | 高热持续，波动<1℃ | 大叶性肺炎 |
| 弛张热 | 高热波动>1℃ | 败血症 |
| 间歇热 | 高热后寒战→反复 | 疟疾 |
| 波状热 | 逐渐升高又逐渐下降 | 布氏杆菌病 |

## 六、应激

### 全身适应综合征（GAS）
| 期 | 时间 | 特点 |
|----|------|------|
| **警觉期** | 立即 | 交感-肾上腺髓质激活，防御准备 |
| **抵抗期** | 持续 | 糖皮质激素主导，适应，但抗感染力↓ |
| **衰竭期** | 持续应激 | 激素耗竭，抵抗↓，应激相关疾病 |

### 应激性溃疡
- 机制：胃黏膜血流↓ + 胃酸分泌↑ + 糖皮质激素↑
- 表现：胃黏膜多发浅表性溃疡
- 预防：PPI、抑酸

## 七、休克

### 分类

| 类型 | 机制 | 例子 |
|------|------|------|
| **低血容量性** | 血容量↓ | 失血、脱水 |
| **心源性** | 心泵功能↓ | 心肌梗死 |
| **分布性** | 血管扩张 | 脓毒症（最常见）|
| **梗阻性** | 血流通道梗阻 | 肺栓塞 |

### 休克进展

**代偿期（早期）**
- 交感激活 → 心率↑、血管收缩
- 维持血压，保证心脑供血

**失代偿期（晚期）**
- 微循环淤血 → 回心血量↓
- DIC → 器官功能障碍
- 不可逆损伤

### 关键指标
```
CVP（中心静脉压）：反映回心血量
MAP = CO × SVR（外周血管阻力）
CI（心脏指数）：正常>2.5 L/min/m²
```

## 八、弥散性血管内凝血（DIC）

### 机制
```
严重感染/创伤/恶性肿瘤
         ↓
凝血系统激活 → 微血栓形成
         ↓
消耗性凝血病（凝血因子↓、血小板↓）
         ↓
继发纤溶 → 出血
```

### 诊断要点
- PT/APTT延长
- 纤维蛋白原↓
- D-二聚体↑
- 血小板↓

## 九、缺血-再灌注损伤

### 机制
```
缺血 → ATP↓ → 次黄嘌呤↑
         ↓
再灌注 + O₂ → 氧自由基爆发
         ↓
ROS损伤（脂质过氧化、蛋白质变性、DNA损伤）
         ↓
钙超载 + 炎症激活
```

### 器官损伤
| 器官 | 表现 |
|------|------|
| 心 | 心肌顿抑、再灌注心律失常 |
| 脑 | 神经元死亡 |
| 肾 | 急性肾损伤 |
| 肝 | 肝细胞坏死 |

## 十、MODS（多器官功能障碍综合征）

### 定义
两个或两个以上器官同时或序贯发生功能障碍

### 常见器官
| 器官 | 指标 |
|------|------|
| 肺 | ARDS（PaO₂/FiO₂<300）|
| 肾 | 肌酐↑、尿量↓ |
| 肝 | 胆红素↑、转氨酶↑ |
| 心 | EF↓、心律失常 |
| 脑 | GCS↓ |
| 凝血 | DIC |

### 常见诱因
- 严重感染（最常见）
- 创伤/大手术
- 休克
- 急性胰腺炎
- 大量输血
