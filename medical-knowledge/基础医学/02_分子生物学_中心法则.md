# 分子生物学：中心法则

## 一、中心法则

```
DNA → RNA → 蛋白质
  ↑      ↓
  ←←←←←←
   逆转录（部分病毒）
```

---

## 二、DNA复制

### 半保留复制
- 每条链作为新链的模板
- DNA聚合酶III（原核）/ DNA Pol α/δ（真核）
- 5'→3' 合成方向

### 复制过程
```
1. 解链（解旋酶 helicase）
2. 引物合成（引物酶 primase）→ DNA聚合酶需要引物
3. 链延伸（DNA Pol III）
4. 片段连接（DNA连接酶 ligase）
```

### 复制叉
- 前导链：连续合成
- 滞后链：不连续合成（冈崎片段）
- DNA Pol I：切除引物，填补缺口
- DNA连接酶：连接片段

### 端粒与端粒酶
- 线性染色体末端：TTAGGG（人类）
- 端粒酶（telomerase）：延伸端粒
- 体细胞：端粒酶阴性 → 复制后端粒缩短 → 复制性衰老
- 肿瘤细胞：端粒酶激活 → 无限增殖

## 三、转录

### RNA聚合酶
| 酶 | 转录 |
|----|------|
| **RNA Pol I** | rRNA（28S, 18S, 5.8S）|
| **RNA Pol II** | mRNA，snRNA，miRNA |
| **RNA Pol III** | tRNA，5S rRNA |

### 真核转录过程
```
1. 转录起始
   - 转录因子（TFIID等）识别启动子（TATA盒）
   - 形成前起始复合物
   - RNA Pol II磷酸化，脱离抑制

2. 链延伸
   - 5'→3' 合成
   - CTD（C端结构域）磷酸化调控

3. 转录终止
   - 加尾信号（AAUAAA）
   - 多聚腺苷酸化（poly A尾）
```

### 前体mRNA加工
- **5'帽**：7-甲基鸟苷（m⁷G），保护+促进翻译
- **剪接**：去除内含子，连接外显子（可变剪接→蛋白多样性）
- **3'多聚A尾**：稳定性+出核

## 四、逆转录

### 逆转录病毒
```
RNA → 逆转录酶 → DNA → 整合入宿主基因组

HIV → 整合前病毒 → 宿主细胞永久感染
```

### 逆转录酶特性
- RNA依赖的DNA聚合酶
- DNA依赖的DNA聚合酶（合成第二链）
- RNase H：降解RNA-DNA杂交中的RNA

## 五、翻译

### 核糖体结构
| | 原核 | 真核 |
|--|------|------|
| 大亚基 | 50S（23S rRNA, 5S rRNA）| 60S（28S, 5.8S, 5S rRNA）|
| 小亚基 | 30S（16S rRNA）| 40S（18S rRNA）|
| 完整 | 70S | 80S |

### 翻译过程

**1. 起始**
```
- 起始tRNA（Met-tRNAi）识别起始密码子AUG
- 原核：30S亚基 + mRNA + fMet-tRNA → 50S结合
- 真核：40S + eIFs → 60S结合
```

**2. 延伸**
```
- 进位（EF-Tu/EF-1）：氨酰-tRNA进入A位
- 成肽（肽酰转移酶）：肽键形成
- 移位（EF-G/EF-2）：核糖体移动，tRNA进入P位
```

**3. 终止**
```
- 终止密码子：UAA, UAG, UGA
- 释放因子（RF1/RF2）识别终止密码子
- 肽链释放
```

## 六、遗传密码

### 密码子表
- 64个密码子 = 61个氨基酸 + 3个终止密码子
- **AUG**：起始密码子 + 甲硫氨酸
- **UAA/UAG/UGA**：终止密码子
- **简并性**：多数氨基酸有≥2个密码子（摆动配对）

### 摆动配对（wobble）
第三位碱基配对不严格
- U可配A/G
- C可配G/U
- A可配U
- G可配C/U

## 七、基因突变

### 点突变
| 类型 | 效应 |
|------|------|
| **同义突变** | 氨基酸不变（简并性）|
| **错义突变** | 氨基酸改变 |
| **无义突变** | 产生终止密码子 |
| **移码突变** | 插入/缺失（非3的倍数）→ 阅读框移位 |

### 突变后果
| 效应 | 例子 |
|------|------|
| **中性突变** | 不影响功能 |
| **有害突变** | 功能下降/丧失 |
| **显性负效应** | 突变蛋白干扰正常蛋白功能 |
| **功能获得** | 新功能/癌基因激活 |

## 八、基因表达调控

### DNA水平
- **染色质重塑**：乙酰化（开放）vs 甲基化（沉默）
- **DNA甲基化**：CpG岛甲基化 → 转录沉默

### 转录水平
- **启动子/增强子**：转录因子结合位点
- **转录因子**：激活或抑制转录
- **沉默子**：负调控元件

### 转录后水平
- **可变剪接**：同一基因→不同蛋白
- **mRNA稳定性**：AU-rich元件（ARE）→ 降解/稳定
- **非编码RNA**：miRNA（抑制翻译/降解mRNA）、lncRNA

### 翻译水平
- **起始因子磷酸化**：eIF2α磷酸化 → 翻译抑制
- **mTOR通路**：营养充足时促进翻译

## 九、DNA修复

| 机制 | 修复对象 |
|------|----------|
| **错配修复（MMR）** | 复制错配 |
| **碱基切除修复（BER）** | 氧化/烷化损伤 |
| **核苷酸切除修复（NER）** | 嘧啶二聚体（UV损伤）|
| **双链断裂修复** |  |
| **同源重组（HR）** | 精确修复（姐妹染色单体）|
| **非同源末端连接（NHEJ）** | 快速但易错 |

### 修复缺陷与肿瘤
- **BRCA1/2**：HR修复关键基因，突变→乳腺癌/卵巢癌
- **MLH1/MSH2**：MMR基因，突变→Lynch综合征
- **XPA-XPG**：NER基因，突变→色素性干皮病（皮肤癌）
