# 免疫学基础

## 一、免疫系统概述

### 固有免疫（innate immunity）
- 第一时间防线
- 无特异性，无记忆
- 屏障、吞噬细胞、自然杀伤细胞、補体、细胞因子

### 适应性免疫（adaptive immunity）
- 第二时间防线
- 高度特异性
- 有记忆（B细胞/T细胞）
- 体液免疫（B细胞/抗体） + 细胞免疫（T细胞）

### 中枢免疫器官
| 器官 | 功能 |
|------|------|
| **骨髓** | 所有血细胞起源，B细胞成熟 |
| **胸腺** | T细胞成熟、选择（TCR多样性生成）|

### 外周免疫器官
| 器官 | 功能 |
|------|------|
| **淋巴结** | 淋巴细胞聚集，抗原呈递 |
| **脾脏** | 过滤血液，免疫细胞居留 |
| **扁桃体** | 黏膜免疫入口 |

## 三、免疫细胞

### 吞噬细胞

**中性粒细胞**
- 占比最高的免疫细胞（50-70%）
- 吞噬+杀菌（呼吸爆发）
- 第一时间募集到感染部位

**巨噬细胞**
- 组织居留型吞噬细胞
- 抗原呈递（ MHC II）
- 组织修复（M2型：促修复；M1型：促炎）
- **枯否细胞**（肝）、**小胶质细胞**（脑）

**树突状细胞（DC）**
- 最强抗原呈递细胞（APC）
- 固有免疫 ↔ 适应性免疫桥梁
- 成熟DC迁移至淋巴结，激活T细胞

### 淋巴细胞

**T细胞**
| 亚群 | 表面标记 | 功能 |
|------|----------|------|
| CD4⁺ Th | TCR αβ + CD4 | 辅助T细胞，分泌细胞因子 |
| CD8⁺ CTL | TCR αβ + CD8 | 细胞毒性，杀死感染/肿瘤细胞 |
| Treg | CD4⁺CD25⁺FoxP3⁺ | 免疫抑制 |
| NKT | TCR NK1.1⁺ | 固有样T细胞，快速细胞毒性 |

**Th亚群分化：**
- Th1：IL-12驱动，IFN-γ主导，细胞内病原体
- Th2：IL-4驱动，IL-4/IL-5/IL-13主导，寄生虫、过敏
- Th17：IL-6+TGF-β驱动，IL-17主导，自身免疫
- Tfh：生发中心，辅助B细胞

**B细胞**
- 表面IgM/IgD（B细胞受体）
- 活化后分化为浆细胞（分泌抗体）
- 记忆B细胞

**自然杀伤细胞（NK）**
- 固有淋巴样细胞（ILC）
- 无TCR，不需抗原致敏
- 杀伤应激/病毒感染/肿瘤细胞
- 机制：KIR识别MHC I，缺失→激活

## 四、抗体（免疫球蛋白）

### 结构
```
Fab区（可变区+铰链区）——抗原结合
Fc区（恒定区）——效应功能
```

### Ig类型与功能

| 类型 | 位置 | 功能 |
|------|------|------|
| **IgM** | 五聚体，血液 | 最早产生，补体激活（经典途径）|
| **IgG** | 血液，组织 | 主力，胎盘转运，ADCC |
| **IgA** | 分泌液，黏膜 | sIgA，黏膜免疫 |
| **IgE** | 肥大细胞表面 | 过敏反应，寄生虫防御 |
| **IgD** | B细胞表面 | B细胞活化辅助 |

### 抗体效应机制
- **中和作用**：阻止病原体黏附/侵入
- **补体激活**：IgG/IgM → 经典途径
- **ADCC**：IgG Fc → NK细胞杀伤
- **调理作用**：促进吞噬细胞吞噬

## 五、补体系统

### 激活途径

| 途径 | 启动物 | C3转化酶 |
|------|--------|----------|
| **经典途径** | 抗原-抗体复合物 | C1→C4b2a |
| **MBL途径** | 甘露糖残基 | MBL→C4b2a |
| **旁路途径** | 微生物表面 | C3bBb |

### 末端效应
- **MAC（膜攻击复合物）**：C5b-9，溶解靶细胞
- **C3a/C5a**：过敏毒素，招募炎症细胞

## 六、主要组织相容性复合体（MHC）

### HLA（人类MHC）

| 类别 | 呈递 | T细胞 |
|------|------|------|
| **HLA-I**（A/B/C）| 内源性肽（病毒、肿瘤）| CD8⁺ CTL |
| **HLA-II**（DR/DQ/DP）| 外源性肽（细菌）| CD4⁺ Th |

### 临床关联
- **移植排斥**：HLA匹配程度决定排斥风险
- **自身免疫**：某些HLA与疾病强关联（HLA-B27与强直性脊柱炎）
- **药物过敏**：HLA-B*5801与别嘌醇SJS/TEN

## 七、免疫应答过程

### 初次应答
1. 抗原进入 → APC摄取处理
2. APC迁移至淋巴结
3. T细胞识别APC的MHC-肽 → 活化
4. B细胞识别抗原 + Th辅助 → 活化
5. 克隆扩增 → 效应细胞/记忆细胞

### 二次应答（记忆）
- 更快（数小时vs数天）
- 更强（更高亲和力抗体）
- 记忆细胞长期存活（数年）

## 九、免疫相关疾病

### 超敏反应（分型）

| 型 | 机制 | 例子 |
|----|------|------|
| **I型（速发型）** | IgE+肥大细胞 | 过敏性休克、哮喘 |
| **II型（细胞毒型）** | IgG+细胞表面抗原 | 自身免疫性溶血性贫血 |
| **III型（免疫复合物型）** | 循环免疫复合物沉积 | SLE、血清病 |
| **IV型（迟发型）** | T细胞介导 | 结核菌素试验、接触性皮炎 |

### 自身免疫病
- **器官特异性**：1型糖尿病、桥本甲状腺炎
- **系统性**：SLE（多系统）、类风湿关节炎（关节）

### 免疫缺陷病
- **原发性（PID）**：SCID（严重联合免疫缺陷）、X-连锁无丙种球蛋白血症
- **继发性**：HIV/AIDS（CD4⁺ T细胞↓）、化疗后

## 十、免疫治疗原理

### 检查点抑制（Immune Checkpoint）
- **PD-1/PD-L1**：T细胞耗竭，肿瘤逃逸
- **CTLA-4**：T细胞活化抑制
- **药物**：Pembrolizumab（PD-1）、Nivolumab、Ipilimumab（CTLA-4）

### CAR-T
```
患者T细胞 → 基因改造（CAR） → 体外扩增 → 回输
CAR = 抗原受体（识别肿瘤）+ 激活域 + 共刺激信号
```

### 疫苗
- **预防性**：HPV疫苗（VLPs）、HBV疫苗
- **治疗性**：肿瘤新抗原疫苗
