# 系统综述与Meta分析

## 一、定义与区别

| 类型 | 定义 | 特点 |
|------|------|------|
| **叙述性综述** | 作者主观汇总文献 | 可能偏倚，无严格方法 |
| **系统综述** | 预先定义方法，系统检索+评价 | 可重复，方法学质量可评估 |
| **Meta分析** | 系统综述中的统计合并方法 | 定量汇总效应量 |

**系统综述可以没有Meta分析（定性系统综述）**

---

## 二、注册与规范

### PROSPERO注册
- 预期注册（先于数据收集）
- 提高透明度和可信度
- 防止报告偏倚（"抽屉问题"）

### PRISMA声明（2020）
- 27项清单 + 流程图
- 必须完整报告每个步骤

### Cochrane系统综述
- 金标准方法学
- 定期更新
- 高质量，偏倚风险低

## 三、系统综述步骤

### 1. 构建问题（PICO）
```
P: Patient/Population（患者人群）
I: Intervention（干预）
C: Comparison（对照）
O: Outcome（结局）
```

### 2. 制定检索策略
**检索数据库（至少）：**
- PubMed/MEDLINE
- EMBASE
- Cochrane Library
- 注册平台（ClinicalTrials.gov, ChiCTR）

**检索词构建：**
```
(干预 OR 疾病) AND (结局) AND (研究类型)
使用MeSH词 + 自由词
布尔逻辑：OR（扩大）、AND（缩小）、NOT（排除）
```

### 3. 筛选文献
**纳入标准预先定义：**
- 研究设计（RCT、队列等）
- 干预/暴露
- 结局
- 语言
- 随访时间

**筛选流程（PRISMA流程图）：**
```
数据库检出 → 去重 → 题目摘要筛选 → 全文评估 → 纳入
```

### 4. 偏倚风险评估

**RCT：Cochrane RoB 2.0**
- 随机化过程
- 偏离预期干预
- 结局数据缺失
- 结局测量
- 选择性报告

**非随机研究：ROBINS-I**
- 混杂
- 受试者选择
- 干预分类
- 偏离干预
- 缺失数据
- 结局测量
- 选择性报告

### 5. 数据提取
- 研究特征（作者、年份、设计）
- 干预/暴露详情
- 结局数据
- 偏倚风险评估

## 四、Meta分析统计方法

### 效应量选择

| 结局类型 | 效应量 |
|----------|--------|
| 二分类 | OR, RR, RD |
| 连续 | MD, SMD |
| 生存 | HR |

### Meta分析模型

**固定效应模型（逆方差法）**
```r
library(meta)
meta_fixed <- metabin(
  event.e = e1, n.e = n1,    # 实验组
  event.c = e2, n.c = n2,    # 对照组
  data = mydata,
  method = "Inverse",
  sm = "OR"
)
```

**随机效应模型（DerSimonian-Laird）**
```r
meta_random <- metabin(
  event.e = e1, n.e = n1,
  event.c = e2, n.c = n2,
  data = mydata,
  method = "Inverse",
  sm = "OR",
  random = TRUE  # 随机效应
)
```

### 合并效应量解读
- 森林图（Forest plot）可视化
- 95%CI不跨越无效线 → 统计学显著

---

## 五、异质性处理

### 异质性检验
```r
# Cochrane Q检验
metabias(meta_random)

# I²
meta_random$I2
```

### 亚组分析
```r
meta_subgroup <- update(meta_random, 
                        subgroup = ~study_type,
                        print.subgroup = TRUE)
```

### Meta回归
```r
metareg(meta_random, ~ age + dosage)
```

### 敏感性分析
- 逐一剔除研究
- 比较固定vs随机效应
- 剔除高偏倚风险研究

## 六、发表偏倚

### 检测方法

**漏斗图（Funnel plot）**
- X轴：效应量
- Y轴：标准误
- 倒漏斗形=无偏倚
- 不对称=可能发表偏倚

**Egger检验**
```r
metabias(meta_random, method = "Egger")
```
P<0.05 = 存在小样本效应

**Trim-and-fill**
- 估计并填补"缺失"研究
- 调整合并效应量

## 七、GRADE证据质量评级

| 质量等级 | 含义 |
|----------|------|
| **高** | 我们非常确信真实效应接近效应量估计 |
| **中** | 对效应估计有中等信心 |
| **低** | 对效应估计信心有限 |
| **极低** | 对效应估计几乎无信心 |

### 降级因素
- 偏倚风险（研究局限性）
- 不一致性（异质性）
- 间接性（人群/干预/结局）
- 不精确性（宽CI）
- 发表偏倚

### 升级因素
- 大效应量
- 剂量-反应梯度
- 混杂因素会降低效应

## 八、诊断准确性Meta分析

### SROC曲线
```
X轴：1-特异度（假阳性率）
Y轴：灵敏度（真阳性率）
```

### 效应量
- **DOR**（诊断优势比）：综合指标
- **灵敏度**、**特异度**：分别合并

### 阈值效应
- 灵敏度和特异度负相关
- SROC曲线非线性
- Spearman相关检验

## 九、网络Meta分析（NMA）

### 基本原理
```
同时比较多个干预（A vs B, A vs C, B vs C）
构建证据网络图
可比较未直接比较的干预
```

### 假设
- **一致性**：直接和间接证据方向一致
- **传递性**：研究间特征可比较

### 方法
- 频率学派：MTC（Mixed Treatment Comparison）
- 贝叶斯学派：WinBUGS/JAGS/Stan

## 十、常见问题

**Q：异质性高怎么办？**
A：不强行合并，先探索原因（亚组/回归），或不做合并

**Q：合并后P<0.05但I²>75%？**
A：以随机效应结果为准，异质性高时P值不可靠

**Q：可以用观察性研究的Meta分析做因果推断吗？**
A：证据等级低（2b-3），需谨慎，只能提供相关性线索
