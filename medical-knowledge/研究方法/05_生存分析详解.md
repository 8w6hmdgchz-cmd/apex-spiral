# 生存分析详解

## 一、核心概念

**生存分析解决的问题：**
- 同时考虑结局（死亡/复发）和时间（到结局的时间）
- 处理截尾数据（Censoring）——部分受试者未观察到结局

**截尾类型：**
| 类型 | 含义 |
|------|------|
| 右截尾 | 对象在研究结束时仍未发生结局 |
| 左截尾 | 对象在进入研究时已经发生结局 |
| 区间截尾 | 知道在某个区间内发生，不知具体时间 |

---

## 二、生存函数

### 定义
```
S(t) = P(T > t)  # 生存时间大于t的概率
```

### Kaplan-Meier法（非参数估计）

**公式：**
```
S(t) = ∏(n_i - d_i) / n_i   （在t_i时刻）

n_i = t_i时刻有风险的个体数
d_i = t_i时刻发生结局的个体数
```

**特点：**
- 每发生一例结局，就更新生存率
- 适合无截尾或少量截尾
- 可计算中位生存时间（S(t)=0.5）

### 生存曲线比较：Log-rank检验

**H0：** 两组生存曲线无差异

**原理：** 比较各时刻观察数与期望数

```r
library(survival)
fit <- survfit(Surv(time, status) ~ group, data = mydata)
survdiff(Surv(time, status) ~ group, data = mydata)
```

**注意事项：**
- χ² 统计量，自由度=组数-1
- P<0.05 → 两组有统计学差异
- 不给出效应量，需补充HR

## 三、Cox比例风险模型

### 模型形式
```
h(t) = h₀(t) × exp(β₁X₁ + β₂X₂ + ...)

h(t): t时刻的风险函数（瞬时事件发生率）
h₀(t): 基线风险函数（X全为0时）
exp(β): 风险比（Hazard Ratio, HR）
```

### 比例风险假设（PH assumption）
- 风险比不随时间变化
- 检验方法：Schoenfeld残差图
- 若违反：需用时变Cox模型或分层Cox

### 模型解读
```
HR = 1：两组风险相同
HR = 2：处理组风险是对照组的2倍
HR = 0.5：处理组风险降低50%
```

### R实现
```r
library(survival)
cox <- coxph(Surv(time, status) ~ treatment + age + stage, data = mydata)
summary(cox)
```

## 五、时依协变量

### 违背PH假设时的处理

**方法1：时变协变量**
- 协变量随时间变化
- Cox模型中用 tt() 函数

**方法2：分层Cox模型**
- 按违背PH的变量分层
- 其他协变量估计HR

**方法3：参数模型**
- 假设基线风险分布（Weibull、指数）
- AIC比较模型拟合

## 八、常见设计

| 设计 | 特点 | 适用场景 |
|------|------|----------|
| 两组平行 | 标准RCT | 比较两种干预 |
| 优效性 | H1: HR<1 | 新药优于对照 |
| 非劣效性 | H1: HR<Δ | 新药不劣于已有治疗 |
| 等效性 | H1: | 新药与对照等效 |
| 剂量爬坡 | 逐步增加剂量 | I期临床 |
| 交叉（Cross-over） | 洗脱期后互换 | 慢性病、症状研究 |

### 非劣效性边界Δ
```
Δ不能大于已有治疗与安慰剂的差值
否则变成"与安慰剂比较"
```

## 九、报告规范

**CONSORT extension for survival outcomes：**
- Kaplan-Meier曲线必须提供
- 中位生存时间及95%CI
- HR及95%CI
- log-rank检验P值
- 截尾比例应报告

**生存曲线解读要点：**
- Y轴：生存概率
- X轴：随访时间
- 曲线高度差 = 生存率差
- 垂直落差 = 某时刻事件发生
- 小标记 = 截尾
