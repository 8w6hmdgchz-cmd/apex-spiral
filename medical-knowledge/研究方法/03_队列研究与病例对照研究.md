# 队列研究与病例对照研究

## 一、队列研究（Cohort Study）

### 基本原理
```
暴露 → 随访 → 结局
━━━━━━━━━━━━━━━━━━━
从暴露到结局，前瞻性观察
```

### 时间方向
| 类型 | 特点 | 优势 | 劣势 |
|------|------|------|------|
| **前瞻性** | 先有暴露，再追结局 | 减少回忆偏倚，可计算发病密度 | 成本高，耗时 |
| **回顾性** | 暴露和结局均已发生 | 快速、成本低 | 依赖记录质量 |
| **双向性** | 混合 | 兼顾两者 | 设计与分析复杂 |

### 结局指标

**发病密度（Incidence Density）**
```
ID = 结局事件数 / 人时（person-time）
单位：事件数/1000人年
```

**相对危险度（RR）**
```
RR = 暴露组发病率 / 非暴露组发病率
```

| RR值 | 意义 |
|------|------|
| =1 | 无关联 |
| >1 | 正向关联（暴露增加风险）|
| <1 | 负向关联（暴露降低风险）|

### 归因危险度（AR）
```
AR = 暴露组发病率 - 非暴露组发病率
```
AR = 暴露组中可归因于暴露的额外风险

### 人时计算
- **Person-years（人年）**：N个人随访Y年 = N×Y人年
- 允许随访时间不同

## 二、病例对照研究（Case-Control Study）

### 基本原理
```
病例（有病）← 回顾 ← 对照（无病）
      ← 暴露史 ←
━━━━━━━━━━━━━━━━━━━━━
从疾病回溯到暴露
```

### 优势与劣势
| 优势 | 劣势 |
|------|------|
| 适合罕见病 | 回忆偏倚 |
| 成本低，耗时短 | 难以计算发病率 |
| 可同时研究多个暴露因素 | 选择对照困难 |

### 比值比（Odds Ratio, OR）
```
OR = (a/c) / (b/d) = ad/bc

病例组暴露比值 / 对照组暴露比值
```

| OR值 | 意义 |
|------|------|
| =1 | 无关联 |
| >1 | 正向关联 |
| <1 | 负向关联 |

**OR与RR的关系（稀有病假设）：**
当疾病发病率<10%时，OR ≈ RR
发病率较高时，OR会高估RR

## 三、研究设计与质量控制

### 队列研究质量要素

**暴露定义**
- 明确、可测量
- 避免错分（misclassification）

**随访完整性**
- 失访率<20%理想
- 失访原因需报告
- 失访与暴露/结局的关联需分析

**结局定义**
- 预先定义
- 客观标准（死亡、发病）
- 盲法评估（减少察觉偏倚）

### 病例对照研究质量要素

**病例来源**
- 社区来源（选择偏倚小）
- 医院来源（选择偏倚大，但代表性）

**对照选择**
- 来自同一源人群
- 匹配（matching）：按重要混杂因素匹配
  - 优点：控制混杂
  - 缺点：匹配变量不能再分析

**回忆偏倚控制**
- 病例可能更努力回忆暴露
- 尽量使用客观记录

## 四、混杂控制

### 混杂因素特征
1. 与暴露相关
2. 是结局的独立危险因素
3. 不是暴露-结局的中间变量

### 控制方法

**设计阶段**
- 随机化（队列研究）
- 限制（restriction）
- 匹配（matching）

**分析阶段**
- 分层分析（Stratification）
- Mantel-Haenszel方法
- 多因素回归（Logistic/Cox）
- **倾向性评分（Propensity Score）**

### 倾向性评分（PS）
```
PS = P(暴露 | 混杂变量) = logit⁻¹(β₀ + β₁X₁ + β₂X₂ + ...)
```

**用途：**
- 当混杂变量多时，比多因素回归更稳定
- 1:1匹配（最常用）
- 分层分析
- 逆概率加权（IPW）

**R实现：**
```r
library(MatchIt)
ps <- matchit(treatment ~ age + sex + stage,
              data = mydata,
              method = "nearest",
              ratio = 1)
matched <- match.data(ps)
```

## 五、Meta分析（合并研究）

### 固定效应 vs 随机效应

**固定效应模型（Fixed-effects）**
- 假设所有研究效应量相同
- 只考虑组内变异
- 只纳入高质量同质研究

**随机效应模型（Random-effects）**
- 假设效应量分布而非单一真值
- 考虑组内+组间变异（τ²）
- 推荐大多数情况

### 异质性检验

**Q检验（Cochran's Q）**
```
H0：所有研究效应量相同
Q = Σwᵢ(Yᵢ - Ŷ)²
服从χ²分布
```

**I²统计量**（更常用）
```
I² = (Q - df) / Q × 100%

I² < 25%：低异质性
25% ≤ I² < 75%：中等异质性
I² ≥ 75%：高异质性
```

### 异质性处理
- 探索来源（亚组分析、Meta回归）
- 随机效应模型
- 敏感性分析（剔除高风险研究）

- 22项清单
- 流程图（入组、失访）
- 混杂因素报告

- 研究设计
- 随访时间和完整性
- 暴露定义
- 结局定义
- RR/AR及95%CI

- 病例和对照来源
- 匹配变量
- 暴露定义和测量
- OR及95%CI
