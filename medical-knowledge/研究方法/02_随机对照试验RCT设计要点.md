# 随机对照试验（RCT）设计要点

## 一、RCT核心特征

| 要素 | 要求 |
|------|------|
| **随机化** | 受试者分配概率均等 |
| **对照** | 阳性对照/安慰剂对照/标准治疗对照 |
| **盲法** | 单盲/双盲/三盲 |
| **前瞻** | 干预在前，结局测量在后 |

**无此四要素 → 不是RCT → 证据等级降级**

---

## 二、随机化方法

### 简单随机（Simple Randomization）
```
每个受试者独立，等概率分配
硬币抛掷、随机数字表、计算机随机数
问题：小样本时组间不均衡风险高
```

### 区组随机（Block Randomization）
```
将受试者分成若干区组，每组内干预/对照比例平衡
例如：区组大小=4 → AABA, ABAB, BAAB, BABA...
优点：保证各时期入组均衡
缺点：若盲法破裂可能推断分组
```

### 分层随机（Stratified Randomization）
```
先按重要预后因素分层，再在每层内随机
例如：按年龄分层（<60/≥60）+ 性别分层
目的：保证分层因素在组间均衡
```

### 最小化法（Minimization）
```
动态分配，优先分配到使整体不均衡最小的组
适合多预后因素、样本量不大的研究
计算机算法实现
```

## 三、对照组设置

| 类型 | 适用场景 | 伦理考量 |
|------|----------|----------|
| **安慰剂对照** | 无标准治疗 | 必须确保不给治疗不造成严重损害 |
| **阳性对照** | 已存在有效治疗 | 需证明新药不劣于/优于已有药 |
| **标准治疗对照** | 金标准已确立 | 伦理上更易接受 |
| **剂量-反应对照** | 探索量效关系 | 多臂设计 |

**注意：** 新药 vs 标准治疗的非劣效性试验需预先设定非劣效性边界（Δ）

---

## 四、盲法设计

| 盲法 | 对象 | 防止偏倚 |
|------|------|----------|
| **单盲** | 受试者不知分组 | 安慰剂效应、依从性 |
| **双盲** | 受试者+研究者不知 | 以上 + 实施偏倚 |
| **三盲** | +统计分析者不知 | 以上 + 分析偏倚 |

**无法盲法的场景：** 外科手术vs保守治疗、行为干预

**补救措施：** 盲法评估（Blind assessment）、结局客观指标

## 五、样本量计算

**主要参数：**
```
n = 2 × (Z_α/2 + Z_β)² × σ² / δ²

α = 0.05（双侧）→ Z_α/2 = 1.96
Power = 0.80 → Z_β = 0.84
Power = 0.90 → Z_β = 1.28
```

**分类变量：**
```r
# R语言 pwr包
library(pwr)
pwr.2p.test(h = 0.3, sig.level = 0.05, power = 0.8)
# h = Cohen's h (效应量)
```

**连续变量：**
```r
pwr.t.test(d = 0.5, sig.level = 0.05, power = 0.8, type = "two.sample")
# d = Cohen's d
```

## 六、主要终点与次要终点

### 主要终点（Primary Endpoint）
- **定义：** 评估干预效果的核心指标，只能有1个
- **选择原则：** 临床相关性高、测量客观、可重复
- **常见错误：** 事后更改主要终点（p-hacking）

### 次要终点（Secondary Endpoint）
- 支持主要终点、探索其他效应
- 不应基于次要终点得出确定性结论

### 替代终点（Surrogate Endpoint）
- 替代临床终点的生物学指标
- 需经过验证（血压降≠心血管死亡降）
- 警惕：肿瘤缩小的替代终点有效性

## 七、常见偏倚与控制

| 偏倚类型 | 控制措施 |
|----------|----------|
| 选择偏倚 | 随机化、分配隐藏 |
| 实施偏倚 | 盲法、标准化干预流程 |
| 测量偏倚 | 盲法评估、客观指标 |
| 随访偏倚 | 标准化随访、ITT分析 |
| 报告偏倚 | 预先注册、结果锁定 |

---

## 八、分配隐藏（Allocation Concealment）

**比盲法更重要！**

| 方法 | 质量 |
|------|------|
| 中央随机（电话/网络） | ★★★★★ |
| 编号容器法 | ★★★★ |
| 顺序编号不透明容器 | ★★★ |
| 开放随机表 | ★★（差） |

**破坏隐藏的方式：** 提前知道下一例分配、分组信封不透明

---

## 九、分析集定义

### 全分析集（Full Analysis Set, FAS）
- 意向性治疗原则（Intention-to-Treat, ITT）
- 纳入所有随机化受试者
- **保守估计，减少偏倚**

### 符合方案集（Per Protocol, PP）
- 纳入完成完整干预的受试者
- 依从性好、无重大方案违背
- **可能高估效应**

### 安全性分析集（Safety Set）
- 至少接受一次干预的受试者

**主要分析应在 FAS 上进行，PP 作为敏感性分析**

## 十一、常见问题

**Q：样本量计算需要预试验数据吗？**
A：可以参考文献中的σ估算，预试验是更可靠的方法

**Q：主要终点能不能在试验中更改？**
A：原则上不能。若需更改必须提前在注册平台更新，说明理由

**Q：中期分析怎么做？**
A：需预先设定α消耗函数（O'Brien-Fleming、Pocock），独立数据安全监查委员会（DSMB）执行
