# 医学文献批判性评价

> 核心原则：不是翻译文献，是找缺陷；不是接受结论，是验证逻辑

---

## 一、评价框架

### 四问法
1. **研究问题清晰吗？** → PICO
2. **方法学质量如何？** → 偏倚风险
3. **结果是什么？** → 效应量+CI
4. **结果适用于我的患者吗？** → 外部效度

---

## 二、RCT评价清单（Cochrane RoB 2.0）

### 1. 随机化过程
- [ ] 随机序列生成是否正确？
- [ ] 分配隐藏是否充分？
- [ ] 组间基线是否可比？

### 2. 偏离预期干预
- [ ] 受试者是否接受了分配的干预？
- [ ] 是否使用了ITT分析？
- [ ] 依从性如何？

### 3. 结局数据缺失
- [ ] 是否存在失访？
- [ ] 失访率是否<20%？
- [ ] 失访是否均衡于两组？
- [ ] 是否进行了敏感性分析？

### 4. 结局测量
- [ ] 是否采用了盲法？
- [ ] 结局测量是否客观？
- [ ] 终点指标是否预先定义？

### 5. 选择性报告
- [ ] 主要终点是否预先注册？
- [ ] 是否报告了所有预设终点？
- [ ] 是否有事后分析未报告？

## 三、队列研究评价（NOS量表）

### selection（选择）
- [ ] 暴露组和非暴露组是否来自同一人群？
- [ ] 暴露定义是否清晰？
- [ ] 是否避免了选择偏倚？

### comparability（可比性）
- [ ] 是否控制了重要混杂因素？
- [ ] 统计方法是否合适？

### outcome（结局）
- [ ] 结局是否客观？
- [ ] 随访时间是否充分？
- [ ] 截尾是否处理得当？

---

## 四、病例对照研究评价

### 关键问题
- **病例来源**：社区 vs 医院（选择偏倚风险不同）
- **对照选择**：是否与病例在同一人群中选取
- **回忆偏倚**：病例是否更易回忆暴露
- **混杂控制**：是否匹配/分层/多因素

## 五、系统综述/Meta分析评价（AMSTAR 2）

### 关键条目
- [ ] 是否预先注册了方案？
- [ ] 检索策略是否全面？（≥2数据库）
- [ ] 是否报告了纳入排除标准？
- [ ] 是否评估了偏倚风险？
- [ ] 异质性是否报告？（I²统计量）
- [ ] 发表偏倚是否评估？（漏斗图/Egger检验）
- [ ] 资金来源是否报告？

### 异质性处理
```
I² < 25%：低异质性，固定效应模型
25% ≤ I² < 75%：中等异质性，随机效应模型
I² ≥ 75%：高异质性，需探索原因
```

## 六、结果解读关键点

### 效应量
**不仅仅看P值！**

| 效应量 | 小 | 中 | 大 |
|--------|-----|-----|-----|
| Cohen's d | 0.2 | 0.5 | 0.8 |
| OR/RR | 1.2 | 1.5 | 2.0 |
| HR | 1.2 | 1.5 | 2.0 |

### 置信区间
- P>0.05 但 CI 很宽 → 证据不足，不能说"无效"
- P<0.05 且 CI 不包含null → 有统计学意义
- CI 反映精确度：越窄越好

### 临床意义
- 统计显著 ≠ 临床重要
- 需要 NNT（需治疗人数）评估
```
NNT = 1 / ARR（绝对风险差）
```

## 八、批判性思维问题清单

### 关于研究问题
- 研究问题是什么？清晰吗？
- 为什么这个问题重要？
- 之前有没有相关研究？

### 关于方法
- 研究设计是否适合回答这个问题？
- 样本量是否足够？
- 随机化/盲法是否充分？
- 混杂因素是否控制？
- 随访是否完整？

### 关于结果
- 效应量有多大？
- 置信区间多宽？
- 结果有临床意义吗？
- 亚组分析是预设还是事后？

### 关于结论
- 作者是否过推？
- 结论是否超出了数据支持范围？
- 有没有替代解释？
- 对我的患者适用吗？

## 九、快速评价口诀

**SOAP for papers：**
- **S**ubjective：研究问题是什么？
- **O**bjective：方法学质量如何？
- **A**ssessment：结果是什么？
- **P**lan：对我的实践有什么意义？
