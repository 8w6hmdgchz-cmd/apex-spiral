# 样本量计算详解

## 一、核心原理

样本量过小 → 检验效能不足 → 错过真实效应  
样本量过大 → 资源浪费 → 伦理问题（让更多人不必要地暴露于风险）

**目标：** 在给定α和Power下，检测到临床有意义的最小效应

---

## 二、两类错误

| 错误类型 | 定义 | 符号 | 通常取值 |
|----------|------|------|----------|
| **I类错误** | 实际无效，但拒绝了H0（假阳性） | α | 0.05 |
| **II类错误** | 实际有效，但拒绝了H1（假阴性） | β | 0.20 |
| **Power** | 正确拒绝H0的概率 | 1-β | 0.80 |

---

## 三、基本公式

### 均数比较（两组平行设计）

```
n = 2 × (Z_α/2 + Z_β)² × σ² / δ²

n = 每组样本量
σ = 标准差（来自预试验或文献）
δ = 期望差异（临床有意义的最小差值）
```

### 两组率比较

```
n = (Z_α/2 + Z_β)² × [p₁(1-p₁) + p₂(1-p₂)] / (p₁-p₂)²

p₁, p₂ = 两组预期率
```

### 配对设计

```
n = (Z_α/2 + Z_β)² × σd² / δ²

σd = 差值的标准差
```

---

## 四、影响样本量的因素

| 因素 | 增大样本量 |
|------|------------|
| α减小（0.05→0.01） | ↑ |
| Power增大（0.80→0.90） | ↑ |
| δ减小（更小效应） | ↑ |
| σ增大（变异增大） | ↑ |

---

## 五、R实现

### 两组均数比较
```r
library(pwr)
pwr.t.test(d = 0.5,         # Cohen's d（效应量）
            sig.level = 0.05,
            power = 0.80,
            type = "two.sample",
            alternative = "two.sided")
```

### 两组率比较
```r
pwr.2p.test(h = 0.3,         # Cohen's h
            sig.level = 0.05,
            power = 0.80,
            alternative = "two.sided")
```

### Cox回归
```r
library(powerSurvEpi)
n <- qsurv(power = 0.80,
            hr = 0.6,        # 期望HR
            alpha = 0.05,
            T = 3,           # 随访时间
            lambda0 = 0.1)   # 基线事件率
```

## 六、非劣效性试验样本量

### 关键区别
- 优效性：H1: HR < 1
- 非劣效性：H1: HR < Δ（非劣效边界）

```r
# 非劣效性
power.nxtest(alpha = 0.025,   # 单侧0.025（非劣效通常单侧）
             m = 200,          # 入组率
             T = 2,           # 随访时间
             lambda0 = 0.1,  # 对照组事件率
             delta = 0.8,     # 非劣效边界（HR < 0.8）
             power = 0.80)
```

### Δ 的设定原则
- 不能大于已有阳性对照 vs 安慰剂的效应
- 需临床和统计学双重论证
- 过大 → 无法证明真正有效

---

## 七、Cluster RCT（整群随机）

### ICC影响
- 组内相关系数（ICC）越大 → 有效样本量越小
- 需用设计效应（Design Effect, DE）校正

```
DE = 1 + (m-1) × ICC
n_adjusted = n_simple × DE
m = 每个cluster的平均样本量
```

## 八、重复测量设计

### 混合效应模型
```r
library(powerLMM)
p <- power_lmm(
  effect = 0.5,     # 效应量
  n_clusters = 20,  # cluster数
  n_subjects = 30,  # 每个cluster人数
  n_times = 4,      # 测量次数
  alpha = 0.05,
  power = 0.80
)
```

## 九、多重性调整

### 多次检验问题
- 多个终点 → α膨胀
- 多个亚组 → α膨胀
- 中期分析 → α消耗

### 调整方法
| 方法 | 原理 | 适用 |
|------|------|------|
| Bonferroni | α/n | 保守，简单 |
| Holm | step-down | 比Bonferroni功效更高 |
| FDR (Benjamini-Hochberg) | 控制伪发现率 | 探索性研究 |
| alpha消耗函数 | O'Brien-Fleming | 中期分析 |

---

## 十、常见错误

1. **δ设置过小** → 样本量天文数字
2. **σ估计不准确** → 用文献但人群不同
3. **忽视脱落** → 实际样本不足
4. **非劣效边界随意** → Δ=0.8不是万能的
5. **忽略ICC** → Cluster RCT常见错误

- 效应量假设及依据
- σ估计来源
