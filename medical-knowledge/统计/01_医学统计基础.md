# 医学统计基础

## 一、描述性统计

### 集中趋势
| 指标 | 适用 | 特点 |
|------|------|------|
| 均数（Mean） | 正态分布 | 受极端值影响 |
| 中位数（Median） | 偏态分布 | 稳健 |
| 众数（Mode） | 分类数据 | |

### 离散程度
| 指标 | 公式 | 特点 |
|------|------|------|
| 标准差（SD） | √(Σ(x-x̄)²/(n-1)) | 正态分布 |
| 四分位距（IQR） | Q3-Q1 | 偏态分布 |
| 方差（Variance） | SD² |  |
| 变异系数（CV） | SD/Mean×100% | 可比性比较 |

## 二、统计推断

### 假设检验原理

**H0（零假设）**：干预无效，差异=0  
**H1（备择假设）**：干预有效，差异≠0（双侧）或>0/<0（单侧）

**决策规则：**
- P ≤ α → 拒绝H0 → 有统计学意义
- P > α → 不拒绝H0 → 无统计学意义

**注意：** "不拒绝"≠"证明H0成立"

### 常用统计方法选择

**连续变量（正态）：**
- 两组：t检验（独立/配对）
- 多组：方差分析（ANOVA）
- 重复测量：重复测量ANOVA

**连续变量（非正态）：**
- 两组：Mann-Whitney U检验
- 多组：Kruskal-Wallis H检验
- 配对：Wilcoxon符号秩检验

**分类变量：**
- 四格表：χ²检验/Fisher精确概率
- 配对四格表：McNemar检验
- 多分类：χ²检验

**相关性：**
- 正态+连续：Pearson相关系数
- 偏态/等级：Spearman相关系数

## 三、效应量（Effect Size）

**为什么重要：**
- p值受样本量影响
- 小样本大效应 ≠ 大样本小效应
- 效应量=临床意义的核心指标

### 主要效应量

| 类型 | 指标 | 含义 |
|------|------|------|
| **Cohen's d** | (x̄₁-x̄₂)/SD_pooled | 0.2=小, 0.5=中, 0.8=大 |
| **OR** | odds₁/odds₀ | 病例对照研究 |
| **RR** | incidence₁/incidence₀ | 队列研究/RCT |
| **HR** | 风险比 | 生存分析 |
| **r** | 相关系数 | 相关分析 |
| **η²** | η² = SS_between/SS_total | 方差分析效应量 |

## 四、回归分析

### 线性回归（连续结局）
```
Y = β₀ + β₁X₁ + β₂X₂ + ... + ε
```

### Logistic回归（二分类结局）
```
logit(P) = ln(P/(1-P)) = β₀ + β₁X₁ + ...
```

### Cox回归（生存结局）
```
h(t) = h₀(t) × exp(β₁X₁ + β₂X₂ + ...)
```

## 五、生存分析

**核心概念：**
- 生存函数 S(t)：t时刻存活概率
- 风险函数 h(t)：t时刻死亡条件概率
- 中位生存期：S(t)=0.5对应的t

**Kaplan-Meier法：**
- 非参数估计
- 适用于无截尾或少量截尾
- log-rank检验比较生存曲线差异

**Cox回归：**
- 半参数模型
- 不要求生存分布
- HR解读：HR=2 → 风险增加一倍

## 六、诊断试验评价

| 指标 | 公式 | 含义 |
|------|------|------|
| 灵敏度（Se） | TP/(TP+FN) | 实际患者检出率 |
| 特异度（Sp） | TN/(TN+FP) | 实际正常人排除率 |
| 阳性预测值（PPV） | TP/(TP+FP) | 阳性结果正确率 |
| 阴性预测值（NPV） | TN/(TN+FN) | 阴性结果正确率 |
| 似然比（LR） | Se/(1-Sp) | 验后概率比 |

**ROC曲线：**
- X轴：1-Sp（假阳性率）
- Y轴：Se（真阳性率）
- AUC：0.5（无诊断）→ 1.0（完美）
- AUC>0.7：可接受
- AUC>0.9：优秀
