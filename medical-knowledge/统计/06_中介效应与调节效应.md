# 中介效应与调节效应

## 一、基本概念区分

| 类型 | 定义 | 问题 |
|------|------|------|
| **中介效应** | X→M→Y（M是X影响Y的机制）| "如何"发生 |
| **调节效应** | Z影响X→Y的方向或强度 | "何时"或"对谁"更强 |

---

## 二、中介效应

### 因果路径
```
X → a → M → b → Y
总效应 = 直接效应 + 间接效应（a×b）
```

### 统计模型
```
模型M：M = aX + e_M
模型Y：Y = c'X + bM + e_Y

间接效应 = a × b
直接效应 = c'
总效应 = c' + ab
```

### 检验方法

**Baron & Kenny 步骤（经典但过时）**
```
1. c显著 → X对Y有总效应
2. a显著 → X对M有作用
3. b显著 → M对Y有作用
4. c'不显著 → 完全中介
5. c'显著但减小 → 部分中介
```

**Sobel检验**
```r
library(bda)
mediation.test(a, b, se_a, se_b)
# a = X→M系数, b = M→Y系数
```

**Bootstrap（推荐）**
```r
library(medmod)
med <- med(support ~ refrmotiv + condition,
           dv = tolerance,
           data = mydata,
           est = "median")
```

### R实现（lavaan）
```r
library(lavaan)

# 写法：~ 表示路径，~~ 表示协方差
model <- '
  # 直接效应
  tolerance ~ c*support + condition
  
  # 中介路径
  refrmotiv ~ a*support
  tolerance ~ b*refmotiv
  
  # 间接效应
  ab := a * b
  total := c + (a * b)
'

fit <- sem(model, data = mydata, se = "bootstrap", bootstrap = 5000)

# 提取结果
parameterEstimates(fit, boot.ci.type = "bca.simple")
```

### 效应量
```
PM = 间接效应 / 总效应  # 中介比例
```

## 三、调节效应

### 概念
调节变量Z影响X→Y关系的方向或强度

### 可视化
```
无调节：  X────────Y
          ↙
           Z（无影响）

调节效应： X────────Y
          ↙↘
           Z（改变斜率）
```

### 检验方法

**交互项检验**
```r
# 中心化后构造交互项
mydata$x_c <- scale(mydata$x, center = TRUE, scale = FALSE)
mydata$z_c <- scale(mydata$z, center = TRUE, scale = FALSE)
mydata$xz <- mydata$x_c * mydata$z_c

# 加入交互项
model <- lm(y ~ x_c + z_c + xz, data = mydata)
summary(model)

# xz显著 → 调节效应显著
```

### 简单斜率分析
```r
library(interactions)
sim_slopes(model, pred = x_c, modx = z_c, johnson_neyman = TRUE)
```

## 四、有调节的中介

### 模型
X通过M影响Y，但这个间接效应被Z调节

### R实现（lavaan）
```r
model <- '
  # 主效应路径
  M ~ a*X
  Y ~ b*M + cprime*X
  
  # 调节X→M的路径
  M ~ a1*X + a2*Z + a3*X*Z
  
  # 间接效应（条件性）
  indirect_xz := a1 * b
  indirect_lowZ := (a1 + a3*(-1)) * b  # Z低时
  indirect_highZ := (a1 + a3*(1)) * b   # Z高时
'

fit <- sem(model, data = mydata, se = "bootstrap", bootstrap = 5000)
```

## 五、混淆 vs 中介

### 关键区别
| 变量 | 与X的关系 | 与Y的关系 | 时间顺序 |
|------|-----------|-----------|----------|
| **混淆变量** | 关联X | 独立导致Y | 不明确 |
| **中介变量** | 被X导致 | 导致Y | X→M→Y |

### 辨别方法
- 时间顺序：X发生前M存在？→混淆
- 干预X后M是否改变？→改变→M是中介
