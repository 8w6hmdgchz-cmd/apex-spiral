## 2. Bug识别

**关键Bug**: ΔG公式逻辑错误
- H_entropy在**分母** → H提升时ΔG反而下降
- 这与APEX设计矛盾：H代表输出可控性（负熵），应正相关
- 物理类比：热力学G = H - TS，熵增降低G；但APEX的H是"负熵"（可控性）

**其他短板**:
- ξ_anti = 0.76（接近0.75阈值，需强化到0.8+）
- ε_repair = 0.70（需实际benchmark验证）
- Φ_positive = 0.71（中等）

---
## 3. 修复动作

**公式修复**: 从除法改为加法
```
旧: ΔG = (Λ×Θ×K×ξ×Ψ×Φ) / (H×T×ε)
新: ΔG = (Λ×Θ×K×ξ×Ψ×Φ) / (T×ε) + H
```

**物理依据**:
- 乘法项 = (Λ×Θ×K×ξ×Ψ×Φ) / (T×ε) → 基础效能
- +H → 可控性增益（负熵贡献）
- 两者相加符合：基础能力 + 优化增益 = 总体效能
